# МойСклад Direct - Быстрый старт 🚀

## Что это?

Автоматическая двусторонняя синхронизация бонусов между вашей онлайн системой и МойСклад POS.

**Работает автоматически:**
- 🛒 Покупка онлайн → бонусы в МойСклад
- 🏪 Покупка в магазине → бонусы в онлайн систему
- 👤 Новый пользователь → автосвязывание по телефону

---

## Установка (5 минут)

### 1. Обновить базу данных

```powershell
# Сгенерировать Prisma Client
npx prisma generate

# Создать миграцию
npx prisma migrate dev --name moysklad_direct_integration
```

### 2. Добавить переменные окружения

В `.env.local`:
```env
MOYSKLAD_ENCRYPTION_KEY=your-strong-random-key-32-chars-minimum
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

Сгенерировать ключ:
```powershell
-join ((48..57) + (65..90) + (97..122) | Get-Random -Count 32 | ForEach-Object {[char]$_})
```

### 3. Запустить

```powershell
yarn dev
```

## Настройка (3 минуты)

### Шаг 1: Получить данные из МойСклад

1. Откройте МойСклад → Настройки → Токены
2. Создайте новый токен с правами на бонусные программы
3. Скопируйте токен

4. Откройте URL админ-панели МойСклад
5. Скопируйте UUID организации из URL (между `/app/` и следующим `/`)

6. Откройте Настройки → Бонусные программы
7. Откройте вашу программу
8. Скопируйте UUID из URL

### Шаг 2: Настроить в UI

1. Откройте: `http://localhost:3000/dashboard/projects/[PROJECT_ID]/integrations/moysklad-direct`

2. Заполните форму:
   - **Account ID**: UUID организации
   - **API Token**: Bearer токен
   - **Bonus Program ID**: UUID бонусной программы
   - **Sync Direction**: "Двусторонняя"
   - **Auto Sync**: ✅ включить
   - **Активна**: ✅ включить

3. Нажмите "Создать"

4. Нажмите "Проверить подключение" → должно быть ✅

### Шаг 3: Настроить webhook в МойСклад

1. Скопируйте **Webhook URL** и **Webhook Secret** из UI

2. Откройте МойСклад → Настройки → Вебхуки

3. Создайте новый webhook:
   - **Тип события**: Бонусная транзакция
   - **URL**: вставьте Webhook URL
   - **Подпись**: включите и вставьте Webhook Secret

4. Сохраните

## Как это работает?

### Автоматическая синхронизация

**Онлайн покупка:**
```
Webhook → BonusService.awardBonus() → автосинхронизация → МойСклад
```

**Офлайн покупка:**
```
МойСклад POS → Webhook → автосинхронизация → наша система
```

**Новый пользователь:**
```
Регистрация → UserService.createUser() → автосвязывание по телефону
```

### Неблокирующая синхронизация

Синхронизация происходит в фоне и **не влияет** на основной процесс:
- ✅ Бонусы начисляются мгновенно
- ✅ Синхронизация происходит асинхронно
- ✅ Ошибки синхронизации логируются, но не блокируют операции

## Troubleshooting

### Синхронизация не работает?

**Проверьте в UI:**
1. Интеграция активна? ✅
2. Auto Sync включен? ✅
3. Есть ошибки в статусе? ❌

**Проверьте логи:**
```powershell
# В терминале dev сервера ищите:
[moysklad-direct-sync]
```

### Webhook не приходит?

**Проверьте в МойСклад:**
1. Webhook URL правильный?
2. Webhook Secret правильный?
3. Тип события "Бонусная транзакция"?
4. Подпись включена?

**Проверьте в терминале:**
```powershell
# Должны видеть запросы:
POST /api/webhook/moysklad-direct/[projectId]
```

### Пользователь не связывается?

**Требования:**
- У пользователя должен быть телефон
- Телефон в формате +7XXXXXXXXXX
- Контрагент с таким телефоном существует в МойСклад

**Проверьте в логах:**
```powershell
# Ищите:
[user-service] Пользователь автоматически связан с МойСклад
```
