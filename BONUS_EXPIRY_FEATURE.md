# Настройка срока действия бонусов

## 📋 Что добавлено

Теперь вы можете настраивать **срок действия бонусов** для каждого проекта через UI.

## 🎯 Где найти настройку

### При создании нового проекта
1. Нажмите "Создать проект" в дашборде
2. Заполните основные поля (название, домен)
3. В разделе "Бонусные настройки" найдите поле **"Срок действия"**
4. Укажите количество дней (по умолчанию 365)

### В настройках существующего проекта
1. Откройте проект в дашборде
2. Перейдите в "Настройки проекта"
3. В разделе "Бонусная система" найдите поле **"Срок действия бонусов (дней)"**
4. Измените значение и нажмите "Сохранить изменения"

## ⚙️ Как это работает

### Автоматический расчет
При начислении бонусов система автоматически устанавливает дату истечения:
```
Дата истечения = Текущая дата + Срок действия (в днях)
```

### Применяется ко всем типам бонусов
- ✅ Приветственные бонусы
- ✅ Бонусы за покупки
- ✅ Реферальные бонусы
- ✅ Ручные бонусы (если не указана кастомная дата)

### Проверка истечения
- При расчете доступного баланса **истекшие бонусы автоматически исключаются**
- Пользователь не сможет потратить истекшие бонусы
- В истории транзакций видны все бонусы с датой истечения

## 📊 Примеры использования

### Стандартная программа лояльности
```
Срок действия: 365 дней (1 год)
```
Бонусы действуют год с момента начисления.

### Краткосрочная акция
```
Срок действия: 30 дней
```
Бонусы мотивируют быстрые повторные покупки.

### Долгосрочная программа
```
Срок действия: 730 дней (2 года)
```
Бонусы действуют дольше для премиум-клиентов.

## 🔧 Технические детали

### База данных
- Поле: `Project.bonusExpiryDays` (тип: Integer)
- По умолчанию: 365 дней
- Диапазон: 1-3650 дней (10 лет)

### API
- Endpoint: `PUT /api/projects/[id]`
- Параметр: `bonusExpiryDays`
- Валидация: минимум 1 день, максимум 3650 дней

### Влияние на существующие бонусы
- ⚠️ Изменение настройки **НЕ влияет** на уже начисленные бонусы
- ✅ Новые бонусы будут использовать обновленное значение
- ✅ Существующие бонусы сохраняют свою дату истечения

## ❓ Часто задаваемые вопросы

**Q: Можно ли сделать бонусы бессрочными?**  
A: В текущей версии минимальный срок — 1 день. Для бессрочных бонусов установите максимальное значение (3650 дней).

**Q: Что происходит с истекшими бонусами?**  
A: Они автоматически исключаются из доступного баланса. Пользователь видит их в истории, но не может потратить.

**Q: Можно ли установить разный срок для разных типов бонусов?**  
A: В текущей версии срок действия единый для всех типов бонусов в рамках проекта.

**Q: Как пользователь узнает о сроке действия бонусов?**  
A: Информация о дате истечения доступна в Telegram боте и через API. Рекомендуется добавить уведомления о скором истечении бонусов.
