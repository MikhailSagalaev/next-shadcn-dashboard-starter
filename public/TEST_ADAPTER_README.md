# TildaAdapter Test Page

## 🧪 Назначение

Тестовая страница для проверки функционала `TildaAdapter` v3.0.0 в изолированной среде.

## 🚀 Запуск

1. Запустите dev сервер:
```bash
yarn dev
```

2. Откройте в браузере:
```
http://localhost:3000/test-tilda-adapter.html
```

## 📋 Что тестируется

### 1. Инициализация
- Создание экземпляра адаптера
- Проверка наличия всех методов
- Инициализация с mock Core

### 2. Работа с корзиной
- `getCartTotal()` - получение суммы корзины
- `observeCart()` - отслеживание изменений с debounce
- Автоматическое обновление при изменении суммы

### 3. Работа с промокодами
- `applyPromocode()` - применение промокода
- `getCurrentPromocode()` - получение текущего промокода
- `clearPromocode()` - очистка промокода
- `setPromocodeFieldVisibility()` - скрытие/показ поля

### 4. Работа с контактами
- `getContactInfo()` - получение email и телефона
- `observeUserInput()` - отслеживание ввода с debounce и валидацией
- Автоматическая валидация email и телефона

### 5. Работа с товарами
- `getAllProducts()` - получение всех товаров на странице
- `initProductBadges()` - добавление бонусных плашек
- `observeProductChanges()` - отслеживание динамических товаров
- Тест добавления товара динамически

### 6. Валидация
- `validateEmail()` - тесты валидации email
- `validatePhone()` - тесты валидации телефона

### 7. Очистка ресурсов
- `destroy()` - корректное освобождение ресурсов
- Отключение всех observers
- Очистка таймеров

## 🎨 Структура страницы

Страница имитирует структуру Tilda:
- `.t706__cartwin` - корзина
- `.t706__cartwin-totalamount` - сумма заказа
- `.t-inputpromocode__wrapper` - поле промокода
- `.t-store__card` - карточки товаров
- `input[name="email"]` - поле email
- `input[name="phone"]` - поле телефона

## 📝 Примечания

- Адаптер работает в debug режиме (все логи включены)
- Mock Core имитирует методы `onPlatformCartUpdate` и `onUserDataUpdate`
- Страница имитирует структуру Tilda, но не является реальным Tilda сайтом
- Для тестирования на реальном Tilda сайте используйте integration тесты
