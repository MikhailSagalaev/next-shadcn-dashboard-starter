# МойСклад Direct - Быстрая настройка (Чеклист)

⏱️ **Время:** 15-30 минут

---

## ✅ Чеклист настройки

### 📋 Подготовка (5 минут)

- [ ] **1.1** Открыть МойСклад: https://online.moysklad.ru/
- [ ] **1.2** Скопировать Account ID из URL:
  ```
  https://online.moysklad.ru/app/#company/[ACCOUNT_ID]
  ```
- [ ] **1.3** Создать API Token:
  - Настройки → Пользователи → Токены → Создать
  - Скопировать токен СРАЗУ!
- [ ] **1.4** Скопировать Bonus Program ID:
  - Настройки → Бонусная программа → Открыть программу
  - Скопировать ID из URL

### 🎯 Создание интеграции (5 минут)

- [ ] **2.1** Открыть UI:
  ```
  https://your-domain.com/dashboard/projects/[ID]/integrations/moysklad-direct
  ```
- [ ] **2.2** Заполнить форму:
  - Account ID: `[вставить]`
  - API Token: `[вставить]`
  - Bonus Program ID: `[вставить]`
  - Sync Direction: `Двусторонняя`
  - Auto Sync: `✅`
  - Is Active: `✅`
- [ ] **2.3** Нажать "Создать"

### ✅ Проверка подключения (2 минуты)

- [ ] **3.1** Нажать "Проверить подключение"
- [ ] **3.2** Убедиться: `✅ Подключение успешно`

### 🔗 Настройка webhook (5 минут)

- [ ] **4.1** Скопировать из UI:
  - Webhook URL
  - Webhook Secret
- [ ] **4.2** Открыть МойСклад → Настройки → Вебхуки
- [ ] **4.3** Создать вебхук:
  - Название: `Bonus System Sync`
  - URL: `[вставить Webhook URL]`
  - Тип события: `Бонусная транзакция`
  - Действие: `Создание`
  - Подпись: `✅` + `[вставить Secret]`
  - Активен: `✅`
- [ ] **4.4** Сохранить

### 🧪 Тестирование (10 минут)

**Тест 1: Онлайн → МойСклад**
- [ ] **5.1** Создать покупку онлайн (Tilda/InSales)
- [ ] **5.2** Проверить логи в UI (должна быть запись `outgoing`)
- [ ] **5.3** Проверить баланс в МойСклад (должен увеличиться)

**Тест 2: МойСклад → Онлайн**
- [ ] **5.4** Создать продажу в МойСклад POS
- [ ] **5.5** Проверить логи в UI (должна быть запись `incoming`)
- [ ] **5.6** Проверить баланс пользователя (должен увеличиться)

### 📊 Проверка логов (3 минуты)

- [ ] **6.1** Открыть "Логи синхронизации" в UI
- [ ] **6.2** Убедиться что есть записи
- [ ] **6.3** Проверить статусы (должны быть `success`)

## 🎉 Готово!

Если все пункты отмечены ✅ - интеграция работает!

## 🚨 Быстрое решение проблем

### ❌ Подключение не удалось
→ Проверьте Account ID и API Token

### ❌ Webhook не приходит
→ Проверьте URL и Secret в МойСклад

### ❌ Пользователь не найден
→ Проверьте формат телефона (+7XXXXXXXXXX)

### ❌ Синхронизация не работает
→ Проверьте Auto Sync и Is Active (должны быть включены)
