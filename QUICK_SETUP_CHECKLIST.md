## 🚀 1. Деплой (2 минуты)

```bash
ssh root@gupil.ru
cd /var/www/gupil
git pull && rm -rf .next && yarn build && pm2 restart bonus-app
```

**Проверка:** `pm2 logs bonus-app --lines 20`

- [ ] Откройте: https://online.moysklad.ru/app/#company/edit
- [ ] Настройки → Токены → Создать новый
- [ ] Права: Чтение + Запись
- [ ] Скопируйте Bearer токен: `_________________`

### ✅ Account ID
- [ ] Ваш email: `_________________`
- [ ] Или из URL: `https://online.moysklad.ru/app/#company/[ACCOUNT_ID]`

### ✅ Bonus Program ID
- [ ] Откройте: https://online.moysklad.ru/app/#discount
- [ ] Найдите бонусную программу
- [ ] Скопируйте UUID из URL: `_________________`

## 🎯 3. Настройте интеграцию (2 минуты)

- [ ] Откройте: `https://gupil.ru/dashboard/projects/[PROJECT_ID]/integrations/moysklad-direct`
- [ ] Заполните форму:
  - Account ID: `_________________`
  - API Token: `_________________`
  - Bonus Program ID: `_________________`
  - Направление: `Двусторонняя`
  - ☑ Автоматическая синхронизация
  - ☑ Активна

- [ ] Нажмите "Тест" → должно быть ✅

## 🔗 4. Настройте Webhook (2 минуты)

- [ ] Откройте: https://online.moysklad.ru/app/#company/edit
- [ ] Настройки → Вебхуки → Создать новый
- [ ] URL: `https://gupil.ru/api/webhook/moysklad-direct/[PROJECT_ID]`
- [ ] События: `Бонусные операции`
- [ ] Метод: `POST`
- [ ] Формат: `JSON`
- [ ] Сохраните

## 🆘 Если что-то пошло не так

### ❌ Тест подключения не прошел
```
1. Проверьте API Token (скопирован полностью?)
2. Проверьте права доступа токена
3. Проверьте Account ID (правильный email?)
```

### ❌ Синхронизация не работает
```
1. Проверьте, что интеграция активна (☑)
2. Проверьте Bonus Program ID (правильный UUID?)
3. Проверьте логи: pm2 logs bonus-app
```

### ❌ Webhook не работает
```
1. Проверьте URL webhook (правильный PROJECT_ID?)
2. Проверьте, что webhook активен в МойСклад
3. Проверьте логи: pm2 logs bonus-app | grep "webhook"
```
