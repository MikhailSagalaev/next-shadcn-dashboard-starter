## ⚡ Быстрый старт

### 1. Деплой (1 команда)
```bash
ssh root@gupil.ru && cd /var/www/gupil && git pull && rm -rf .next && yarn build && pm2 restart bonus-app
```

### 2. Настройка (3 параметра)
- API Token → МойСклад: Настройки → Токены
- Account ID → Ваш email
- Bonus Program ID → UUID из URL

### 3. Запуск (3 клика)
1. Заполните форму в админ-панели
2. Нажмите "Тест" → ✅
3. Настройте Webhook в МойСклад
