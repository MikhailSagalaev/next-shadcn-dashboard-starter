## 🚀 Как использовать

### 1. Импорт workflow

```bash
# Файл находится в корне проекта
Система лояльности без подписки (шаблон).json
```

**Шаги:**
1. Откройте Dashboard → Projects → [ваш проект] → Workflow
2. Нажмите "Импортировать"
3. Выберите файл `Система лояльности без подписки (шаблон).json`
4. Подтвердите импорт

### 2. Настройка проекта

Убедитесь, что в настройках проекта указаны:

```typescript
{
  welcomeBonus: 100,              // Сумма приветственных бонусов
  bonusExpiryDays: 365,           // Срок действия (1 год)
  bonusMode: 'SIMPLE',            // Режим начисления
  bonusPercentage: 5.0,           // Процент кешбэка (для SIMPLE)
  bonusBehavior: 'SPEND_AND_EARN' // Поведение бонусов
}
```

### 3. Тестирование

**Запустите бота и протестируйте:**

1. Отправьте `/start` боту
2. Должно прийти приветственное сообщение с кнопками:
   - 📱 Поделиться контактом
   - ✉️ Ввести email
3. Поделитесь контактом
4. Должно прийти сообщение об успешной активации с балансом

## 🔍 Важно: Визуализация в конструкторе

### Почему показывается только триггер?

Это **нормальное поведение** конструктора workflow:

- При импорте показываются только **триггеры** (точки входа)
- Остальные ноды появляются при выполнении workflow
- Это сделано для упрощения навигации

### Как проверить, что всё на месте?

**Вариант 1: Через PowerShell**
```powershell

Write-Host "Нод: $($json.nodes.Count)" -ForegroundColor Green
Write-Host "Соединений: $($json.connections.Count)" -ForegroundColor Green
```

**Вариант 2: Запустить бота**
Лучший способ - протестировать реальный флоу с ботом.

## 🆚 Сравнение версий

### С подпиской (оригинал)

**Флоу:**
```
/start → Welcome → Подписка на канал → Контакт → Активация
```

**Плюсы:**
- Собирает подписчиков канала
- Дополнительный канал коммуникации

**Минусы:**
- Дополнительный барьер входа
- Ниже конверсия
- Сложнее для пользователя

### Без подписки (новая)

**Флоу:**
```
/start → Welcome → Контакт → Активация
```

**Плюсы:**
- Быстрая активация
- Выше конверсия
- Проще для пользователя
- Меньше шагов

**Минусы:**
- Не собирает подписчиков канала

## 🐛 Troubleshooting

### Бонусы не начисляются

**Проверьте:**
1. `welcomeBonus > 0` в настройках проекта
2. `welcomeRewardType === 'BONUS'`
3. Логи выполнения workflow
4. Логи бота

### Бот не отвечает

**Проверьте:**
1. Бот запущен (`yarn bot:start` или PM2)
2. Bot token корректный
3. Webhook настроен (если используется)
4. Логи бота на ошибки

### Welcome message не приходит

**Проверьте:**
1. Workflow активирован
2. Триггер `/start` настроен правильно
3. Связь `check-user-status → welcome-message` существует
4. Логи выполнения workflow
