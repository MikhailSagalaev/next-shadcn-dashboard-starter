```tsx
<div className='space-y-2'>
  <Label htmlFor='bonusExpiryDays'>
    Срок действия бонусов (дней)
  </Label>
  <Input
    id='bonusExpiryDays'
    type='number'
    min='1'
    max='3650'
    step='1'
    value={formData.bonusExpiryDays}
    onChange={(e) =>
      setFormData({
        ...formData,
        bonusExpiryDays: parseInt(e.target.value) || 365

    placeholder='365'
  />
```

## 🔄 Как это работает

### При создании проекта
1. Пользователь указывает срок действия бонусов (или оставляет 365 по умолчанию)
2. Значение сохраняется в `Project.bonusExpiryDays`

### При начислении бонусов
```typescript
// Автоматический расчет даты истечения
const expiresAt = new Date(
  Date.now() + project.bonusExpiryDays * 24 * 60 * 60 * 1000
);

// Создание бонуса
await db.bonus.create({
  data: {
    userId,
    amount,
    type: 'PURCHASE',
    expiresAt, // ← Дата истечения
    // ...
  }
});
```

### При расчете баланса
```typescript
// Истекшие бонусы автоматически исключаются
const availableBonuses = await db.bonus.findMany({
  where: {
    userId,
    isUsed: false,
    OR: [
      { expiresAt: null }, // Бессрочные
      { expiresAt: { gt: new Date() } } // Не истекшие
    ]
  }
});
```

## ⚠️ Важные замечания

### Влияние на существующие бонусы
- Изменение настройки **НЕ влияет** на уже начисленные бонусы
- Существующие бонусы сохраняют свою дату истечения
- Новые бонусы будут использовать обновленное значение

### Бессрочные бонусы
В текущей версии минимальный срок — 1 день. Для максимально долгого срока используйте 3650 дней (10 лет).

### Уведомления
Рекомендуется добавить уведомления пользователям о скором истечении бонусов (за 7-14 дней).
