# Быстрый старт

Руководство по созданию проекта и настройке бонусной программы.

## Подготовка

Вам понадобится:
- Доступ к административной панели
- Сайт или интернет-магазин
- Telegram аккаунт

## 1. Создание проекта

1. Войдите в административную панель.
2. Нажмите **"Создать проект"**.
3. Заполните поля:
   - **Название проекта**: название вашего магазина или сервиса.
   - **Домен сайта**: адрес вашего сайта.
   - **Описание**: краткая информация.
4. Нажмите **"Создать"**.

## 2. Настройка бонусов

1. Откройте **"Настройки"** проекта.
2. В разделе бонусной системы укажите:
   - **Режим работы**: `SPEND_AND_EARN` (начисление и списание).
   - **Процент начисления**: например, 10%.
   - **Срок действия**: например, 365 дней.
   - **Мин. сумма начисления**: 100 рублей.
3. В настройках активации включите "Автоматическая активация" и укажите приветственный бонус.

## 3. Telegram бот

1. Создайте бота через @BotFather в Telegram (`/newbot`).
2. Скопируйте полученный **токен**.
3. В Gupil перейдите в **"Telegram бот"**.
4. Вставьте токен и нажмите **"Сохранить и активировать"**.
5. Настройте приветственное сообщение и команды (`/start`, `/balance`, `/help`).

## 4. Интеграция

1. В разделе **"Интеграция"** скопируйте **Webhook URL**.
2. Настройте отправку данных с вашего сайта на этот URL (регистрация, покупки).
3. (Опционально) Добавьте JS-виджет баланса на сайт.

## 5. Тестирование

1. Запустите бота (`/start`).
2. Используйте **"Тест webhook"** в разделе интеграции для эмуляции регистрации и покупок.
3. Убедитесь, что бонусы начисляются, а бот присылает уведомления.
