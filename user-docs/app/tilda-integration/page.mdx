# Интеграция с Tilda

Полное руководство по подключению бонусной системы Gupil к сайтам на Tilda.

> **Новая версия виджета!** Теперь виджет автоматически определяет платформу Tilda и подключает нужные компоненты. Интеграция стала еще проще!

## Быстрый старт

### Шаг 1: Получите код интеграции

1. Войдите в админ-панель Gupil
2. Перейдите в **Проекты** → Выберите ваш проект
3. Откройте вкладку **Интеграция**
4. Выберите платформу **Tilda**
5. Скопируйте код подключения виджета

### Шаг 2: Добавьте код на сайт

1. В Tilda: **Настройки сайта** → **Еще** → **HTML-код для вставки внутри HEAD**
2. Вставьте скопированный код:

```html
<!-- Gupil Bonus Widget -->
<script>
  window.gupilConfig = {
    projectId: 'YOUR_PROJECT_ID',
    apiUrl: 'https://gupil.ru/api'
    // platform: 'tilda' - автоматически определяется
  };
</script>
<script src="https://gupil.ru/widget-loader.js" async></script>
```

3. Сохраните и опубликуйте сайт

> **Как это работает?** Widget Loader автоматически определяет что вы используете Tilda и загружает специальный Tilda Adapter для работы с вашим сайтом.

### Шаг 3: Настройте Webhook

1. В Gupil: **Интеграция** → Скопируйте **Webhook URL**
2. В Tilda: **Настройки сайта** → **Формы** → **Webhook для заказов**
3. Вставьте URL, формат: **JSON**
4. Сохраните и опубликуйте все страницы

**Готово!** Виджет автоматически определит Tilda и начнет работать.

## Что умеет виджет на Tilda?

### ✅ Отображение бонусов на товарах

Виджет автоматически добавляет плашки с бонусами на все товары в каталоге.

**Поддерживаемые блоки каталогов:**
- T776 - Каталог товаров (Grid)
- T754 - Каталог товаров (List)
- T750 - Каталог товаров (Compact)
- T762 - Каталог товаров (Cards)
- T1200+ - Все новые блоки каталогов

**Пример:**
```
┌─────────────────────┐
│  Товар: 5000₽      │
│  [+250 бонусов]    │ ← Автоматически добавляется
└─────────────────────┘
```

### ✅ Применение промокодов

Клиенты могут использовать промокоды для получения скидок.

**Как это работает:**
1. Клиент вводит промокод в поле корзины
2. Виджет проверяет промокод через API
3. Если промокод валиден → применяется скидка
4. Скидка отображается в корзине

### ✅ Списание бонусов при оплате

Клиенты могут оплачивать часть заказа бонусами.

**Как это работает:**
1. Виджет показывает доступные бонусы в корзине
2. Клиент выбирает сколько бонусов использовать
3. Сумма заказа уменьшается
4. При оформлении заказа бонусы списываются

### ✅ Регистрация пользователей

Виджет автоматически регистрирует новых пользователей.

**Как это работает:**
1. Клиент вводит email/телефон в форме оформления
2. Виджет отправляет данные в Gupil API
3. Создается новый пользователь (если не существует)
4. Начисляются приветственные бонусы (если настроено)

### ✅ Автозаполнение контактов

Для постоянных клиентов виджет автоматически заполняет контактные данные.

## Детальная настройка

### Настройка корзины Tilda

Для полной работы виджета нужно включить некоторые опции в корзине:

1. **Настройки сайта** → **Корзина и оплата**
2. Включите опции:
   - ✅ **Поле для промокода** (для применения промокодов)
   - ✅ **Сбор email** (для регистрации пользователей)
   - ✅ **Сбор телефона** (опционально, для Telegram бота)
3. Сохраните изменения

### Настройка Webhook

Webhook нужен для синхронизации заказов с Gupil.

**Что отправляется через Webhook:**
- Новые заказы (для начисления бонусов)
- Данные клиентов (для регистрации)
- Использованные промокоды (для списания бонусов)

**Настройка:**
1. В Gupil: **Интеграция** → Скопируйте **Webhook URL**
   - Формат: `https://gupil.ru/api/webhook/YOUR_SECRET`
2. В Tilda: **Настройки сайта** → **Формы** → **Webhook для заказов**
3. Вставьте URL
4. Формат передачи данных: **JSON**
5. Сохраните

**Проверка:**
- Оформите тестовый заказ
- В Gupil: **Логи** → Проверьте что пришел webhook
- В Gupil: **Пользователи** → Проверьте что клиент зарегистрирован

### Кастомизация виджета

Вы можете настроить внешний вид и поведение виджета:

```html
<script>
  window.gupilConfig = {
    projectId: 'YOUR_PROJECT_ID',
    apiUrl: 'https://gupil.ru/api',
    
    // Опциональные настройки
    theme: 'light',        // Тема: light | dark | auto
    language: 'ru',        // Язык: ru | en
    debug: false,          // Debug режим (для отладки)
    
    // Callbacks
    onReady: function(widget) {
      console.log('Виджет готов!');
    },
    onError: function(error) {
      console.error('Ошибка виджета:', error);
    }
  };
</script>
<script src="https://gupil.ru/widget-loader.js" async></script>
```

## Проверка работы

### 1. Проверка виджета

Откройте сайт и проверьте:

✅ **Плашки с бонусами на товарах**
- Откройте страницу каталога
- На каждом товаре должна быть плашка "+X бонусов"

✅ **Поле промокода в корзине**
- Добавьте товар в корзину
- Откройте корзину
- Должно быть поле для ввода промокода

✅ **Отображение бонусов в корзине**
- В корзине должна быть информация о доступных бонусах
- Должна быть возможность использовать бонусы

### 2. Проверка Webhook

Оформите тестовый заказ:

1. Добавьте товар в корзину
2. Заполните форму оформления
3. Оформите заказ

Проверьте в Gupil:

✅ **Логи** → Должен прийти webhook `purchase`
```json
{
  "action": "purchase",
  "payload": {
    "email": "test@example.com",
    "amount": 5000,
    "orderId": "12345"
  }
}
```

✅ **Пользователи** → Клиент должен быть зарегистрирован
- Email/телефон из формы
- Начислены бонусы за покупку

✅ **Транзакции** → Должна быть запись о начислении
- Тип: "Покупка"
- Сумма: 5% от заказа (или ваш процент)

### 3. Проверка промокодов

Создайте тестовый промокод:

1. В Gupil: **Промокоды** → **Создать промокод**
2. Код: `TEST123`
3. Скидка: 10%

Проверьте на сайте:

1. Добавьте товар в корзину
2. Введите промокод `TEST123`
3. Нажмите "Применить"
4. Скидка должна примениться

## Troubleshooting

### Виджет не загружается

**Проблема:** Виджет не появляется на сайте.

**Решение:**
1. Откройте консоль браузера (F12)
2. Проверьте наличие ошибок
3. Убедитесь что `projectId` указан правильно
4. Проверьте что скрипт загружается: Network → `widget-loader.js`

**Debug режим:**
```javascript
window.gupilConfig = {
  projectId: 'YOUR_PROJECT_ID',
  apiUrl: 'https://gupil.ru/api',
  debug: true // Включить debug логи
};
```

### Бонусы не отображаются на товарах

**Проблема:** Плашки с бонусами не появляются на карточках товаров.

**Решение:**
1. Проверьте что используется поддерживаемый блок каталога (T776, T754, T750, T762)
2. Проверьте что товары загружены в каталог Tilda
3. Проверьте настройки проекта в админ-панели Gupil
4. Откройте консоль и проверьте логи виджета

### Промокод не применяется

**Проблема:** При вводе промокода ничего не происходит.

**Решение:**
1. Проверьте что промокод активен в админ-панели
2. Проверьте что промокод не истек
3. Проверьте лимиты использования промокода
4. Проверьте консоль на наличие ошибок API

### Webhook не приходит

**Проблема:** После оформления заказа webhook не приходит в Gupil.

**Решение:**
1. Проверьте что Webhook URL указан правильно в Tilda
2. Проверьте что формат данных: **JSON**
3. Проверьте что сайт опубликован после настройки webhook
4. Оформите тестовый заказ и проверьте логи в Gupil
