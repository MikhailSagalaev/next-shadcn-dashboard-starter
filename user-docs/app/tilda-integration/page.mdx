# Интеграция с Tilda

Руководство по подключению Gupil к сайтам на Tilda.

## 1. Настройка Webhook

1. В Gupil: **Интеграция** -> Скопируйте **Webhook URL**.
2. В Tilda: **Настройки сайта** -> **Формы** -> **Webhook**.
3. Вставьте URL, формат передачи данных: **JSON**.
4. Сохраните изменения и опубликуйте все страницы.

Теперь при заполнении форм (корзина) данные будут отправляться в Gupil.

## 2. Виджет баланса

Показывает бонусы клиента на сайте.

1. В Gupil: **Интеграция** -> Скопируйте код виджета.
2. В Tilda: Добавьте блок **T123 (HTML)** в футер (или хедер).
3. Вставьте код виджета.

Код:
```html
<script src="https://gupil.ru/widget.js"></script>
<script>
  GupilWidget.init({
    projectId: 'ВАШ_ID',
    apiKey: 'ВАШ_KEY'
  });
</script>
```

## 3. Промокоды (списание)

Позволяет списывать бонусы через поле промокода в корзине.

1. В Tilda: **Настройки корзины** -> **Промокоды** -> Включить поле.
2. Добавьте скрипт валидации (см. полную документацию API), который будет отправлять введенный промокод в Gupil API `/validate-promo`.
3. Gupil вернет размер скидки, эквивалентный бонусам.

## Проверка

1. Оформите тестовый заказ.
2. Проверьте **Логи** в Gupil: должен прийти webhook `purchase`.
3. Проверьте **Пользователи**: клиенту должны начислиться бонусы.
