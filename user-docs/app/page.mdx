# Добро пожаловать

**Gupil** — SaaS платформа для управления бонусными программами с интеграцией Telegram ботов.

## О платформе

Gupil позволяет бизнесу внедрить систему лояльности с автоматическим начислением бонусов, интеграцией с сайтами и Telegram ботами.

## Возможности

### Бонусная система
- Гибкие правила начисления и списания
- Многоуровневая система лояльности
- Автоматическое сгорание бонусов
- Реферальная программа

### Telegram боты
- Конструктор сценариев
- Шаблоны
- Автоматические уведомления

### Интеграция
- Webhook для сайтов и CRM
- REST API
- Виджеты

### Аналитика
- Статистика по клиентам
- Отчеты эффективности
- Экспорт данных
