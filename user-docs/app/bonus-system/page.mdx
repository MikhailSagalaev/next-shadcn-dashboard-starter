# Бонусная система

Руководство по управлению программой лояльности.

## Принципы работы

1. **Регистрация** — начисление приветственных бонусов.
2. **Покупка** — начисление процента от суммы.
3. **Оплата** — списание бонусов.
4. **Срок действия** — сгорание неиспользованных бонусов.

## Настройка

Раздел: `/settings`

### Параметры
- **Процент начисления**: Доля от суммы заказа в бонусах.
- **Срок действия**: Период в днях до сгорания.
- **Приветственный бонус**: Начисление за регистрацию.

### Режимы работы (BonusBehavior)

- **EARN_ONLY**: Только накопление.
- **SPEND_ONLY**: Только списание (актуально при закрытии программы).
- **SPEND_AND_EARN**: Одновременное накопление и списание (базовый режим).

*При режиме SPEND_AND_EARN бонусы начисляются только на часть суммы, оплаченную деньгами.*

## Уровни лояльности

Раздел: `/bonus-levels`

Позволяет менять процент начисления в зависимости от общей суммы покупок клиента.

| Уровень | Мин. сумма | % бонусов | Макс. оплата |
|---------|------------|-----------|--------------|
| Базовый | 0₽ | 5% | 10% |
| Серебряный | 50,000₽ | 10% | 25% |
| Золотой | 150,000₽ | 15% | 40% |

Система автоматически пересчитывает уровень при каждой покупке.

## Реферальная программа

Раздел: `/referral`

Многоуровневая система вознаграждений за приглашение пользователей.

**Параметры:**
- Проценты для уровней (1-3 уровня).
- Приветственный бонус рефералу.
- Срок действия реферальной связи (cookie).

## Управление бонусами

Раздел: `/users`

### Операции администратора
- **Начисление**: Ручное добавление бонусов (подарок, компенсация).
- **Списание**: Ручное уменьшение баланса.
- **Корректировка**: Установка точного значения баланса.

### История транзакций
Типы операций: `EARNED` (покупка), `SPEND` (трата), `EXPIRED` (сгорание), `WELCOME` (регистрация), `REFERRAL` (реферал), `ADMIN` (ручная).
