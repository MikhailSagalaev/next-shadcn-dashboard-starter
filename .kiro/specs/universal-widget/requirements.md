# Universal Widget - Требования к рефакторингу

> **Статус:** 🚧 В разработке  
> **Приоритет:** 🟡 Средний  
> **Дата создания:** 2026-01-31

## 🎯 Цель

Завершить рефакторинг виджета на универсальную архитектуру с адаптерами для поддержки множества платформ (Tilda, Shopify, WooCommerce, custom сайты).

## 📐 Текущая архитектура

```
┌─────────────────────────────────────┐
│  tilda-bonus-widget.js (LEGACY)     │  ← Монолитный, 2000+ строк
│  - Вся логика в одном файле         │
│  - Tilda-специфичный код            │
│  - Дублирование функционала         │
└─────────────────────────────────────┘
```

## 🎨 Целевая архитектура

```
┌─────────────────────────────────────┐
│  universal-widget.js (CORE)         │  ← Платформо-независимое ядро
│  - LeadWidgetCore class             │
│  - API запросы                      │
│  - Управление состоянием            │
│  - Бизнес-логика                    │
└──────────────┬──────────────────────┘
               │
               ↓ использует IWidgetAdapter
┌──────────────┴──────────────────────┐
│  Адаптеры платформ                  │
├─────────────────────────────────────┤
│  tilda-adapter.js                   │  ← Tilda специфика
│  - getCartTotal()                   │
│  - getContactInfo()                 │
│  - applyPromocode()                 │
│  - observeCart()                    │
│  - initProductBadges()              │
├─────────────────────────────────────┤
│  shopify-adapter.js (будущее)       │  ← Shopify специфика
├─────────────────────────────────────┤
│  woocommerce-adapter.js (будущее)   │  ← WooCommerce специфика
├─────────────────────────────────────┤
│  custom-adapter.js (будущее)        │  ← Для custom сайтов
└─────────────────────────────────────┘
```

## 📋 Интерфейс IWidgetAdapter

Каждый адаптер должен реализовывать:

```typescript
interface IWidgetAdapter {
  // Инициализация
  init(): void;
  
  // Получение данных платформы
  getCartTotal(): number;
  getContactInfo(): { email: string | null, phone: string | null };
  
  // Управление промокодами
  applyPromocode(code: string): boolean;
  setPromocodeFieldVisibility(hidden: boolean): void;
  
  // Наблюдатели
  observeCart(): void;
  observeUserInput(): void;
  
  // Бонусные плашки на товарах
  initProductBadges(settings: object, calculatorFn: Function): void;
  
  // Монтирование UI
  mountInlineWidget(renderCallback: Function): void;
  
  // Очистка
  destroy(): void;
}
```
