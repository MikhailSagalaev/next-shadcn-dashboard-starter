## 🎯 Следующие шаги

### 1. Настройка интеграции (5 минут)

Следуйте инструкции: `QUICK_SETUP_CHECKLIST.md`

Кратко:
1. Получите API Token из МойСклад
2. Получите Account ID (ваш email)
3. Получите Bonus Program ID (UUID)
4. Заполните форму в админ-панели
5. Проверьте подключение (кнопка "Тест")

### 2. Настройка Webhook (2 минуты)

1. Откройте: https://online.moysklad.ru/app/#company/edit
2. Настройки → Вебхуки → Создать новый
3. URL: `https://gupil.ru/api/webhook/moysklad-direct/[PROJECT_ID]`
4. События: Бонусные операции
5. Метод: POST, Формат: JSON

### 3. Тестирование (3 минуты)

1. Нажмите "Синхронизировать" в админ-панели
2. Проверьте статистику (должна быть > 0)
3. Проверьте логи синхронизации
4. Проверьте логи на сервере
