## 🔍 Проверка корректности

### Через PowerShell
```powershell
$json = Get-Content "Система лояльности без подписки (шаблон).json" -Raw | ConvertFrom-Json

Write-Host "Нод: $($json.nodes.Count)" -ForegroundColor Green
Write-Host "Соединений: $($json.connections.Count)" -ForegroundColor Green

# Проверка ключевых нод
$keyNodes = @("start-trigger", "welcome-message", "check-contact-user", 
              "activate-user", "add-welcome-bonus", "success-activated-user")

foreach ($nodeId in $keyNodes) {
    $exists = $json.nodes | Where-Object { $_.id -eq $nodeId }
    $status = if ($exists) { "✅" } else { "❌" }
    Write-Host "$status $nodeId"
}
```

## 🐛 Troubleshooting

### Проблема: Бонусы не начисляются

**Решение:**
1. Проверьте `welcomeBonus > 0`
2. Проверьте `welcomeRewardType === 'BONUS'`
3. Посмотрите логи workflow
4. Посмотрите логи бота

### Проблема: Бот не отвечает

**Решение:**
1. Проверьте, что бот запущен
2. Проверьте bot token
3. Проверьте webhook (если используется)
4. Посмотрите логи бота

### Проблема: Welcome message не приходит

**Решение:**
1. Проверьте, что workflow активирован
2. Проверьте связь `check-user-status → welcome-message`
3. Посмотрите логи выполнения
4. Протестируйте с другим пользователем
