Настроить синхронизацию бонусов между вашей онлайн-системой и МойСклад для единого баланса бонусов в онлайн и офлайн каналах.

## Шаг 3: Получите Account ID

### Вариант 1: Из URL

1. Откройте: https://online.moysklad.ru/app/#company/edit
2. Посмотрите на URL в адресной строке
3. Найдите часть после `#company/`

**Пример:**
```
URL: https://online.moysklad.ru/app/#company/a.churova@yandex.ru
Account ID: a.churova@yandex.ru
```

### Вариант 2: Ваш email

Если в URL нет явного Account ID, используйте ваш email для входа в МойСклад.

**Пример:**
```
Email для входа: a.churova@yandex.ru
Account ID: a.churova@yandex.ru
```

## Шаг 4: Получите Bonus Program ID

### 4.1 Откройте список скидок

1. Перейдите: https://online.moysklad.ru/app/#discount
2. Найдите вашу бонусную программу в списке

**Что вы увидите:**
```
┌─────────────────────────────────────────┐
│ Скидки и бонусные программы             │
├─────────────────────────────────────────┤
│ Название              Тип      Статус   │
│ Моя бонусная программа Бонусы  Активна  │ ← Нажмите
│ Скидка 10%            Скидка   Активна  │
└─────────────────────────────────────────┘
```

### 4.2 Скопируйте UUID из URL

1. Нажмите на бонусную программу
2. Посмотрите на URL в адресной строке
3. Скопируйте UUID после `id=`

**Пример:**
```
URL: https://online.moysklad.ru/app/#discount/edit?id=ffd2feee-bee8-11f0-0a80-0311000ca7a6
Bonus Program ID: ffd2feee-bee8-11f0-0a80-0311000ca7a6
```

## Шаг 5: Настройте интеграцию в Gupil

### 5.1 Откройте страницу интеграции

1. Перейдите: https://gupil.ru/dashboard
2. Выберите ваш проект
3. Перейдите: Интеграции → МойСклад Direct

**URL будет выглядеть так:**
```
https://gupil.ru/dashboard/projects/[PROJECT_ID]/integrations/moysklad-direct
```

### 5.2 Заполните форму

**Что вы увидите:**
```
┌─────────────────────────────────────────────────────┐
│ Настройки интеграции                                │
├─────────────────────────────────────────────────────┤
│ API Credentials                                     │
│ ┌─────────────────────────────────────────────────┐ │
│ │ Account ID: [                                 ] │ │
│ │ API Token:  [                                 ] │ │
│ │ Bonus Program ID: [                           ] │ │
│ └─────────────────────────────────────────────────┘ │
│                                                     │
│ Настройки синхронизации                             │
│ ┌─────────────────────────────────────────────────┐ │
│ │ Направление: [Двусторонняя ▼]                  │ │
│ │ ☑ Автоматическая синхронизация                 │ │
│ │ ☑ Активна                                      │ │
│ └─────────────────────────────────────────────────┘ │
│                                                     │
│ [💾 Создать]                                        │
└─────────────────────────────────────────────────────┘
```

**Заполните:**
1. **Account ID**: вставьте ваш email (из Шага 3)
2. **API Token**: вставьте Bearer токен (из Шага 2)
3. **Bonus Program ID**: вставьте UUID (из Шага 4)
4. **Направление**: выберите "Двусторонняя (онлайн ↔ офлайн)"
5. **Автоматическая синхронизация**: включите ☑
6. **Активна**: включите ☑

### 5.3 Сохраните настройки

Нажмите кнопку "Создать"

**Что вы увидите:**
```
✅ Успешно сохранено
Настройки МойСклад Direct интеграции созданы
```

### 5.4 Проверьте подключение

Нажмите кнопку "Тест"

**Что вы должны увидеть:**
```
✅ Подключение успешно
Соединение с МойСклад API установлено
```

**Если видите ошибку:**
- Проверьте API Token (скопирован полностью?)
- Проверьте Account ID (правильный email?)

## ✅ Готово!

Интеграция настроена и работает!

### Что происходит теперь?

1. **Автоматическая синхронизация** - бонусы синхронизируются автоматически при операциях
2. **Webhook** - МойСклад отправляет уведомления о бонусных операциях
3. **Единый баланс** - бонусы доступны в онлайн и офлайн каналах

### Как проверить работу?

1. Начислите бонусы в МойСклад → проверьте баланс в онлайн-системе
2. Начислите бонусы в онлайн-системе → проверьте баланс в МойСклад
3. Проверьте логи синхронизации в админ-панели
