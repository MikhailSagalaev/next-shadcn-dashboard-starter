## 🔄 Как работает синхронизация

### Сценарий 1: Онлайн → МойСклад

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│   Покупка    │────▶│    Наша      │────▶│   МойСклад   │
│   онлайн     │     │   система    │     │     API      │
│ (Tilda/      │     │              │     │              │
│  InSales)    │     │ BonusService │     │ accrueBonus()│
└──────────────┘     └──────┬───────┘     └──────────────┘
                            │
                            ▼
                     ┌──────────────┐
                     │  Sync Log    │
                     │  (outgoing)  │
                     └──────────────┘

Шаги:
1. Пользователь покупает онлайн
2. Webhook → BonusService.awardBonus()
3. Бонусы начислены в нашей системе ✅
4. Автоматически → SyncService.syncBonusAccrualToMoySklad()
5. МойСклад API → accrueBonus()
6. Бонусы синхронизированы в МойСклад ✅
7. Создан sync log
```

### Сценарий 2: МойСклад → Онлайн

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│   Покупка    │────▶│   МойСклад   │────▶│    Наша      │
│   в POS      │     │   Webhook    │     │   система    │
│              │     │              │     │              │
│              │     │ POST /webhook│     │ BonusService │
└──────────────┘     └──────┬───────┘     └──────────────┘
                            │
                            ▼
                     ┌──────────────┐
                     │  Sync Log    │
                     │  (incoming)  │
                     └──────────────┘

Шаги:
1. Пользователь покупает в POS
2. МойСклад создает bonus transaction
3. МойСклад отправляет webhook → наш сервер
4. WebhookHandler → SyncService.syncFromMoySklad()
5. Бонусы начислены в нашей системе ✅
6. Создан sync log
7. Отправлено Telegram уведомление (если настроено)
```

## 🚀 Быстрый старт

1. **Получить данные из МойСклад** (5 мин)
   - Account ID, API Token, Bonus Program ID

2. **Создать интеграцию в UI** (5 мин)
   - Заполнить форму, нажать "Создать"

3. **Проверить подключение** (1 мин)
   - Нажать "Проверить подключение"

4. **Настроить webhook** (5 мин)
   - Скопировать URL и Secret
   - Создать webhook в МойСклад

5. **Протестировать** (10 мин)
   - Создать покупку онлайн
   - Создать продажу в POS
   - Проверить логи

**Готово! ✅**
