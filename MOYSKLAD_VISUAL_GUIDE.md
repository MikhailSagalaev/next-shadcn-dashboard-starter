## 🔍 Troubleshooting

### ❌ "Invalid token"
```
Проблема: Неверный API токен
Решение:
1. Проверьте, что токен скопирован полностью
2. Убедитесь, что токен активен в МойСклад
3. Проверьте права доступа токена
```

### ❌ "Bonus program not found"
```
Проблема: Бонусная программа не найдена
Решение:
1. Проверьте UUID бонусной программы
2. Убедитесь, что программа активна
3. Проверьте права доступа к программе
```

### ❌ "User not found"
```
Проблема: Пользователь не найден
Решение:
1. Убедитесь, что пользователь существует в обеих системах
2. Проверьте соответствие email/телефона
3. Проверьте логи синхронизации
```
