### 5. API - Webhook обработка

**Endpoint:** `POST /api/insales/webhook/[projectId]`

**Тест 1: Создание клиента (clients/create)**

**Payload:**
```json
{
  "id": "clients/create",
  "client": {
    "id": 12345,
    "name": "Иван Иванов",
    "email": "ivan@example.com",
    "phone": "+79991234567",
    "registered_date": "2026-03-04T10:00:00Z"
  }
}
```

**Ожидаемый результат:**
- ✅ Пользователь создан в БД
- ✅ Приветственные бонусы начислены (если настроено)
- ✅ Webhook лог создан
- ✅ HTTP 200 ответ

**Тест 2: Создание заказа (orders/create)**

**Payload:**
```json
{
  "id": "orders/create",
  "order": {
    "id": 67890,
    "client": {
      "id": 12345,
      "email": "ivan@example.com"
    },
    "items_price": "5000.00",
    "full_total_price": "5000.00",
    "discount_amount": "0.00",
    "created_at": "2026-03-04T11:00:00Z"
  }
}
```

**Ожидаемый результат:**
- ✅ Бонусы начислены (5% от 5000 = 250 бонусов)
- ✅ Транзакция создана (type: EARN)
- ✅ Webhook лог создан
- ✅ HTTP 200 ответ

**Проверки:**

### 13. Обработка ошибок

**Тесты:**

1. **Неверный Webhook Secret:**
   - Отправить webhook с неправильным secret
   - Ожидается: HTTP 401 Unauthorized

2. **Неактивная интеграция:**
   - Деактивировать интеграцию в админке
   - Отправить webhook
   - Ожидается: HTTP 403 Forbidden

3. **Невалидный payload:**
   - Отправить webhook с неполными данными
   - Ожидается: HTTP 400 Bad Request + детали ошибки

4. **Несуществующий проект:**
   - Запрос к `/api/insales/balance/non-existent-id`
   - Ожидается: HTTP 404 Not Found

5. **Ошибка InSales API:**
   - Неверные credentials в настройках
   - Попытка применить промокод
   - Ожидается: HTTP 500 + лог ошибки
