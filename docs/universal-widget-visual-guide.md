## 🔄 Как это работает: Пошаговый сценарий

### Сценарий: Пользователь добавляет товар в корзину на Tilda

```
┌─────────────────────────────────────────────────────────────┐
│  1. Клиент добавил виджет на свой Tilda сайт                │
│                                                             │
│  <script>                                                   │
│    window.LEAD_WIDGET_CONFIG = {                            │
│      projectId: 'abc123',                                   │
│      apiUrl: 'https://bonus.gupil.ru'                       │
│    };                                                       │
│  </script>                                                  │
│  <script src="https://bonus.gupil.ru/widget-loader.js">    │
│  </script>                                                  │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│  2. widget-loader.js загружается и работает                 │
│                                                             │
│  ✅ Определяет платформу: Tilda                             │
│  ✅ Загружает universal-widget.js                           │
│  ✅ Загружает tilda-adapter.js                              │
│  ✅ Создает экземпляры:                                     │
│     const core = new LeadWidgetCore(config);                │
│     const adapter = new TildaAdapter(core);                 │
│     core.setAdapter(adapter);                               │
│  ✅ Инициализирует: await core.init();                      │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│  3. Инициализация завершена                                 │
│                                                             │
│  adapter.init()                                             │
│  ├─ Захватывает элементы Tilda DOM                          │
│  ├─ Сохраняет оригинальные стили промокода                  │
│  └─ Готов к работе                                          │
│                                                             │
│  adapter.observeCart()                                      │
│  ├─ Создает MutationObserver на .t706__cartwin             │
│  ├─ Debounce 400ms для оптимизации                          │
│  └─ Следит за изменениями корзины                           │
│                                                             │
│  adapter.observeUserInput()                                 │
│  ├─ Слушает события input/change                            │
│  ├─ Debounce 500ms для оптимизации                          │
│  └─ Следит за вводом email/phone                            │
│                                                             │
│  core.renderUI()                                            │
│  └─ Рендерит виджет на странице                             │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│  4. Пользователь добавляет товар в корзину                  │
│                                                             │
│  [Пользователь кликает "Купить"]                            │
│                          ↓                                  │
│  Tilda обновляет DOM:                                       │
│  .t706__cartwin-totalamount: "5 000 ₽" → "7 500 ₽"         │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│  5. adapter.observeCart() обнаруживает изменение            │
│                                                             │
│  MutationObserver срабатывает                               │
│  ├─ Debounce 400ms (ждет завершения изменений)              │
│  ├─ Вызывает adapter.getCartTotal()                         │
│  │  └─ Парсит .t706__cartwin-totalamount → 7500             │
│  ├─ Сравнивает с lastCartTotal (5000 !== 7500)              │
│  └─ Вызывает core.onPlatformCartUpdate(7500)                │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│  6. Core обрабатывает изменение корзины                     │
│                                                             │
│  core.onPlatformCartUpdate(7500)                            │
│  ├─ Обновляет state: setState({ cartTotal: 7500 })          │
│  ├─ Уведомляет подписчиков через pub/sub                    │
│  ├─ Пересчитывает доступные бонусы                          │
│  └─ Вызывает core.updateUI()                                │
│     └─ Обновляет виджет на странице                         │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│  7. Пользователь видит обновленный виджет                   │
│                                                             │
│  ┌─────────────────────────────────────────────────┐       │
│  │  💰 Ваш баланс: 500 бонусов                     │       │
│  │                                                 │       │
│  │  🎁 Можете списать до 500₽ на этот заказ       │       │
│  │                                                 │       │
│  │  ✨ Начислим 750 бонусов за покупку             │       │
│  │     (10% от 7500₽)                              │       │
│  │                                                 │       │
│  │  [Применить 500 бонусов]                        │       │
│  └─────────────────────────────────────────────────┘       │
└─────────────────────────────────────────────────────────────┘
```

## 📊 Сравнение: Добавление новой платформы

### ❌ БЫЛО (Legacy)

```
Задача: Добавить поддержку Shopify

Шаг 1: Скопировать tilda-bonus-widget.js (2000 строк)
        ↓
Шаг 2: Найти все Tilda-специфичные места (~800 строк)
        ├─ document.querySelector('.t706__cartwin')
        ├─ window.tcart
        ├─ .t-inputpromocode
        └─ ... еще 50+ мест
        ↓
Шаг 3: Заменить на Shopify-специфичный код
        ├─ window.Shopify.cart
        ├─ fetch('/cart.js')
        ├─ Shopify.onCartUpdate()
        └─ ... переписать 800 строк
        ↓
Шаг 4: Получить shopify-bonus-widget.js (2000 строк)
        ↓
Шаг 5: Тестировать вручную на Shopify сайте
        ↓
Шаг 6: Исправлять баги
        ↓
Шаг 7: Повторное тестирование

Результат:
⏱️  Время: 2 недели (80 часов)
📦 Код: 4000 строк (2000 Tilda + 2000 Shopify)
🐛 Баги: Много (нет тестов)
🔧 Поддержка: Кошмар (дублирование кода)
💰 Стоимость: Высокая
```

### ✅ СТАЛО (Universal Widget)

```
Задача: Добавить поддержку Shopify

Шаг 1: Создать shopify-adapter.js (300 строк)
        ↓
Шаг 2: Реализовать IWidgetAdapter интерфейс
        ├─ init() { /* Shopify инициализация */ }
        ├─ getCartTotal() { return Shopify.cart.total_price; }
        ├─ getContactInfo() { return Shopify.customer; }
        ├─ applyPromocode(code) { /* Shopify API */ }
        └─ observeCart() { Shopify.onCartUpdate(...); }
        ↓
Шаг 3: Добавить определение в widget-loader.js (10 строк)
        SHOPIFY: {
          name: 'shopify',
          adapter: 'shopify-adapter.js',
          className: 'ShopifyAdapter',
          detect: () => typeof window.Shopify !== 'undefined'
        }
        ↓
Шаг 4: Написать unit тесты (100 строк)
        describe('ShopifyAdapter', () => {
          it('должен получать сумму корзины', () => {
            window.Shopify = { cart: { total_price: 5000 } };
            expect(adapter.getCartTotal()).toBe(5000);
          });
        });
        ↓
Шаг 5: Тесты проходят автоматически ✅
        ↓
Шаг 6: Готово! Core работает без изменений

Результат:
⏱️  Время: 1 день (8 часов)
📦 Код: 910 строк (600 Core + 300 Shopify + 10 loader)
🐛 Баги: Минимум (покрыто тестами)
🔧 Поддержка: Легко (нет дублирования)
💰 Стоимость: Низкая

Экономия: 72 часа (90% времени)
```
