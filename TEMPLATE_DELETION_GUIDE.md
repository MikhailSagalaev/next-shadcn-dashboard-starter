# Руководство по удалению шаблонов рассылок

## Что изменилось

Теперь вы можете **удалять шаблоны рассылок** прямо из интерфейса!

## Где доступно удаление

### 1. Диалог отправки уведомлений
**Путь:** Проект → Пользователи → Выбрать пользователей → Отправить уведомление

**Как удалить:**
1. Выберите шаблон из выпадающего списка "Выберите шаблон..."
2. Под списком появится красная кнопка **"Удалить выбранный шаблон"**
3. Нажмите на кнопку
4. Подтвердите удаление в диалоге
5. Готово! Шаблон удален

### 2. Редактор workflow (конструктор автоматизаций)
**Путь:** Проект → Workflow → Открыть workflow → Добавить/выбрать ноду "Сообщение"

**Как удалить:**
1. Кликните на ноду "Сообщение"
2. В панели свойств справа выберите шаблон из списка
3. Под списком появится красная кнопка **"Удалить выбранный шаблон"**
4. Нажмите на кнопку
5. Подтвердите удаление
6. Готово! Шаблон удален

## Особенности

✅ **Безопасность:** Перед удалением система попросит подтверждение  
✅ **Уведомления:** После удаления появится уведомление об успехе  
✅ **Автообновление:** Список шаблонов обновится автоматически  
✅ **Умный UI:** Кнопка появляется только когда выбран шаблон  

## Важно знать

⚠️ **Удаление необратимо** - восстановить шаблон будет невозможно  
⚠️ **Проверьте использование** - убедитесь, что шаблон не используется в активных workflow  
⚠️ **Права доступа** - удалять шаблоны могут только администраторы проекта  

## Что делать, если кнопка не появляется

1. Убедитесь, что вы выбрали шаблон из списка
2. Проверьте, что у вас есть права администратора проекта
3. Обновите страницу (F5)
4. Если проблема сохраняется, обратитесь в поддержку

## Создание нового шаблона

Если вы случайно удалили шаблон, вы всегда можете создать новый:

1. Введите текст сообщения в редакторе
2. Нажмите кнопку **"Сохранить шаблон"** (иконка дискеты)
3. Введите название шаблона
4. Нажмите "Сохранить"
