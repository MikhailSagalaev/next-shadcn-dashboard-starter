# 🚀 Быстрый старт: Workflow без подписки

## За 5 минут

### 1. Импорт (1 мин)

```
Dashboard → Projects → [проект] → Workflow → Импортировать
→ Выбрать: "Система лояльности без подписки (шаблон).json"
```

### 2. Настройки (2 мин)

```
Dashboard → Projects → [проект] → Settings
```

Установите:
```typescript
welcomeBonus: 100           // Приветственные бонусы
bonusExpiryDays: 365        // Срок действия (1 год)
bonusMode: 'SIMPLE'         // Простой режим
bonusPercentage: 5.0        // 5% кешбэк
```

### 3. Тест (2 мин)

1. Запустите бота (если не запущен):
   ```powershell
   yarn bot:start
   ```

2. Отправьте `/start` боту в Telegram

3. Нажмите "📱 Поделиться контактом"

4. Проверьте сообщение об успехе с балансом

## ✅ Готово!

Если всё работает - можете использовать в production.

## 🔍 Что изменилось?

**Было (с подпиской):**
```
/start → Подписка на канал → Контакт → Активация
```

**Стало (без подписки):**
```
/start → Контакт → Активация
```
