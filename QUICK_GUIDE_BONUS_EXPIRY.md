# 🚀 Быстрый гайд: Настройка срока действия бонусов

## Где найти настройку?

### Вариант 1: В настройках существующего проекта

```
1. Откройте дашборд → Выберите проект
2. Перейдите в "Настройки проекта"
3. Прокрутите до раздела "Бонусная система"
4. Найдите поле "Срок действия бонусов (дней)"
5. Измените значение (от 1 до 3650 дней)
6. Нажмите "Сохранить изменения"
```

### Вариант 2: При создании нового проекта

```
1. Нажмите "Создать проект"
2. Заполните название и домен
3. В разделе "Бонусные настройки" найдите "Срок действия"
4. Укажите количество дней (по умолчанию 365)
5. Нажмите "Создать проект"
```

## 💡 Популярные настройки

### Стандартная программа лояльности
```
Срок действия: 365 дней
Подходит для: Обычных магазинов, долгосрочных программ
```

### Краткосрочная акция
```
Срок действия: 30-90 дней
Подходит для: Сезонных акций, стимулирования быстрых покупок
```

### Премиум программа
```
Срок действия: 730 дней (2 года)
Подходит для: VIP-клиентов, дорогих товаров
```

### Максимальный срок
```
Срок действия: 3650 дней (10 лет)
Подходит для: Практически бессрочных бонусов
```

## ⚙️ Как работает?

```
Пользователь совершает покупку
         ↓
Система начисляет бонусы
         ↓
Дата истечения = Сегодня + Срок действия
         ↓
Бонусы доступны до даты истечения
         ↓
После истечения бонусы недоступны
```

## ❓ Частые вопросы

**Q: Что происходит с уже начисленными бонусами при изменении настройки?**  
A: Ничего. Существующие бонусы сохраняют свою дату истечения. Новая настройка применяется только к новым бонусам.

**Q: Можно ли сделать бонусы бессрочными?**  
A: Установите максимальное значение 3650 дней (10 лет).

**Q: Как пользователь узнает о сроке действия?**  
A: Информация доступна в Telegram боте (команда /balance) и через API.

**Q: Можно ли установить разный срок для разных типов бонусов?**  
A: В текущей версии — нет. Срок единый для всех типов бонусов в проекте.

**Q: Что видит пользователь при истечении бонусов?**  
A: Истекшие бонусы автоматически исключаются из доступного баланса. В истории они видны, но недоступны для использования.
