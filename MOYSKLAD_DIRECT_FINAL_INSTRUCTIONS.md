## 🎯 Шаг 2: Получить данные из МойСклад (5 минут)

### 2.1 Account ID

1. Откройте: https://online.moysklad.ru/
2. Войдите в аккаунт
3. Скопируйте UUID из URL:
   ```
   https://online.moysklad.ru/app/#company/[ACCOUNT_ID]
   ```
4. Сохраните в блокнот

### 2.2 API Token

1. МойСклад → **Настройки** → **Пользователи и права**
2. Выберите своего пользователя
3. Вкладка **Токены** → **Создать токен**
4. Название: `Bonus System Integration`
5. **ВАЖНО:** Скопируйте токен СРАЗУ (больше не покажется!)
6. Сохраните в блокнот

### 2.3 Bonus Program ID

1. МойСклад → **Настройки** → **Бонусная программа**
2. Если нет программы - создайте:
   - **Создать бонусную программу**
   - Название: `Основная программа`
   - Настройте правила начисления
   - Сохраните
3. Откройте программу
4. Скопируйте UUID из URL:
   ```
   https://online.moysklad.ru/app/#bonusprogram/[BONUS_PROGRAM_ID]
   ```
5. Сохраните в блокнот


## 🧪 Шаг 6: Протестировать синхронизацию (10 минут)

### Тест 1: Онлайн → МойСклад

**Цель:** Проверить, что покупка онлайн синхронизируется в МойСклад

**Шаги:**

1. **Создать тестовую покупку через Tilda/InSales**
   - Используйте тестовый webhook
   - Или создайте реальную покупку

2. **Проверить логи в UI:**
   - Откройте страницу интеграции
   - Секция **"Логи синхронизации"**
   - Найдите запись: `operation: bonus_accrual`, `direction: outgoing`
   - Статус должен быть: `success` ✅

3. **Проверить в МойСклад:**
   - Откройте МойСклад
   - **Контрагенты** → найдите по телефону
   - Проверьте баланс бонусов (должен увеличиться)

**Ожидаемый результат:**
```
✅ Бонусы начислены в нашей системе
✅ Бонусы синхронизированы в МойСклад
✅ Баланс в МойСклад обновлен
```

### Тест 2: МойСклад → Онлайн

**Цель:** Проверить, что покупка в POS синхронизируется в нашу систему

**Шаги:**

1. **Создать транзакцию в МойСклад POS:**
   - Откройте МойСклад POS (кассовое приложение)
   - Создайте продажу для контрагента
   - Начислите бонусы (например, 100 бонусов)
   - Завершите продажу

2. **Проверить webhook:**
   - МойСклад автоматически отправит webhook

3. **Проверить логи в UI:**
   - Откройте страницу интеграции
   - Секция **"Логи синхронизации"**
   - Найдите запись: `operation: bonus_accrual`, `direction: incoming`
   - Статус должен быть: `success` ✅

4. **Проверить в вашей системе:**
   - Откройте профиль пользователя
   - Проверьте баланс бонусов (должен увеличиться)
   - Проверьте историю транзакций

**Ожидаемый результат:**

## 🔍 Troubleshooting

### Проблема 1: "Подключение не удалось"

**Причины:**
- Неправильный Account ID
- Неправильный API Token
- Токен не имеет прав

**Решение:**
1. Проверьте Account ID (должен быть UUID)
2. Проверьте API Token (скопирован полностью?)
3. В МойСклад проверьте права токена

### Проблема 2: "Webhook не приходит"

**Причины:**
- Webhook URL неправильный
- Webhook Secret неправильный
- Webhook не активен в МойСклад

**Решение:**
1. Проверьте URL (должен быть доступен извне)
2. Проверьте Secret (скопирован полностью?)
3. В МойСклад проверьте статус webhook (должен быть активен)

### Проблема 3: "Пользователь не найден"

**Причины:**
- Телефон пользователя не совпадает с МойСклад
- Контрагент не существует в МойСклад

**Решение:**
1. Проверьте формат телефона (должен быть +7XXXXXXXXXX)
2. Создайте контрагента в МойСклад с таким телефоном
3. Или обновите телефон пользователя в вашей системе

### Проблема 4: "Синхронизация не работает"

**Причины:**
- Auto Sync выключен
- Интеграция не активна
- Sync Direction не позволяет

**Решение:**
1. Проверьте Auto Sync (должен быть включен)
2. Проверьте Is Active (должен быть включен)
3. Проверьте Sync Direction (должен быть BIDIRECTIONAL)
