## 🔧 Техническая реализация

### Способ 1: Через интерфейс InSales (рекомендуется)

1. Войдите в админ-панель InSales
2. Перейдите в **Расширения** (НЕ "Настройки"!)
3. Найдите раздел **Webhooks** или **Приложения**
4. Создайте webhook с параметрами выше
5. Активируйте webhook

### Способ 2: Через InSales API

Если в интерфейсе нет возможности создать webhooks, используйте API:

**Создание webhook для orders/create:**
```bash
curl -X POST "https://avocadoshop.myinsales.ru/admin/webhooks.json" \
  -H "Content-Type: application/json" \
  -u "[API_KEY]:[API_PASSWORD]" \
  -d '{
    "webhook": {
      "address": "https://gupil.ru/api/insales/webhook/cmilhq0y600099e7uraiowrmt",
      "topic": "orders/create",
      "format_type": "json"
    }
  }'
```

**Создание webhook для clients/create:**
```bash
curl -X POST "https://avocadoshop.myinsales.ru/admin/webhooks.json" \
  -H "Content-Type: application/json" \
  -u "[API_KEY]:[API_PASSWORD]" \
  -d '{
    "webhook": {
      "address": "https://gupil.ru/api/insales/webhook/cmilhq0y600099e7uraiowrmt",
      "topic": "clients/create",
      "format_type": "json"
    }
  }'
```

**Проверка созданных webhooks:**
```bash
curl -X GET "https://avocadoshop.myinsales.ru/admin/webhooks.json" \
  -H "Content-Type: application/json" \
  -u "[API_KEY]:[API_PASSWORD]"
```

**Примечание:** API_KEY и API_PASSWORD берутся из раздела **Настройки** → **API** в InSales.

## 🧪 Тестирование

### Тест 1: Проверка webhooks

1. Создайте тестовый заказ в магазине
2. Проверьте в админке InSales: **Расширения** → **Webhooks** → История
3. Должен быть запрос с статусом 200 OK
4. Проверьте в Gupil Dashboard: должна появиться запись в логах webhooks

### Тест 2: Проверка виджета

1. Откройте сайт в режиме инкогнито
2. Зарегистрируйтесь как новый клиент
3. Кликните на виджет в правом нижнем углу
4. Должно открыться окно с балансом бонусов
5. Откройте карточку товара - должен быть бейдж "Получите X бонусов"

### Тест 3: Проверка начисления бонусов

1. Создайте заказ на 5000₽
2. Оплатите заказ (или измените статус на "Оплачен")
3. Проверьте баланс в виджете - должно начислиться 500 бонусов (10%)
4. Проверьте в Gupil Dashboard - должна быть транзакция

## 🐛 Возможные проблемы и решения

### Проблема: Webhook не срабатывает

**Решение:**
- Проверьте URL - должен быть точно `https://gupil.ru/api/insales/webhook/cmilhq0y600099e7uraiowrmt`
- Проверьте, что webhook активен
- Проверьте формат - должен быть JSON
- Проверьте логи в InSales - там должен быть статус ответа

### Проблема: Виджет не отображается

**Решение:**
- Проверьте, что код вставлен перед `</body>`
- Откройте консоль браузера (F12) - проверьте ошибки JavaScript
- Проверьте, что скрипт загружается: `https://gupil.ru/insales-widget-loader.js`
- Проверьте `data-project-id` - должен быть `cmilhq0y600099e7uraiowrmt`

### Проблема: Бонусы не начисляются

**Решение:**
- Проверьте статус оплаты заказа - должен быть "Оплачен"
- Проверьте логи webhooks в Gupil Dashboard
- Проверьте, что webhook `orders/create` активен
- Проверьте процент начисления в настройках интеграции (должен быть 10%)
